# JuaKali Konnect

## Overview
JuaKali Konnect is a platform that connects skilled Kenyan artisans ("fundis") with clients who need services like plumbing, welding, carpentry, and more. It helps local workers find jobs and makes it easy for people to hire trusted professionals.

## Features
- **Search for Fundis**: Find artisans by skill and location.
- **Fundi Profiles**: View ratings, experience, and contact info.
- **Secure Payments**: Simulated escrow system using M-Pesa.
- **Fundi Dashboard**: Artisans can manage their profiles and portfolios.
- **Client Profiles**: Manage your account and activity.
- **Responsive Design**: Works on desktop and mobile devices.

## How to Use
1. **For Clients**:
   - Visit the home page and search for a service.
   - Browse available fundis on the "Hire Fundis" page.
   - Contact a fundi directly or use the escrow demo.

2. **For Fundis**:
   - Sign up on the "Hire Fundis" page.
   - Manage your profile in the Fundi Dashboard.
   - Add photos of your work to attract clients.

## Technologies Used
- **HTML**: Structure of the website.
- **CSS**: Styling and layout.
- **JavaScript**: Interactive features and data storage.

## Setup
This is a static website with no server required.
1. Download or clone the project files.
2. Open `index.html` in your web browser.
3. Navigate the website.
